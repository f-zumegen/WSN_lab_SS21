◆ RESET_SQN_NO

#define RESET_SQN_NO   10

Used to implement lollipop-shaped sequence number spaces. Hybrid of linear and circular sequence number spaces. That is the value from which we start our sequnce number space when first going live, or a node is reset/down. When a sequence number hits 255 it circulated to 0 with a modulo oeration. Then when we generate a new packet and attach the sequence number 1, the receiver checks that 1 <= RESET_SQN_NO, recognizing the shift, taking over value and updating the old sequence number.
